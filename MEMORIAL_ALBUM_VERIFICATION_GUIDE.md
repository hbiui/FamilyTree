## 1. 功能概述

### 1.1 主要功能
- ✅ **海报模板选择**：提供 4 种风格模板（经典典雅、现代简约、极简黑白、喜庆红金）
- ✅ **成员选择**：支持选择全部或部分家族成员
- ✅ **拖拽布局**：可自由拖拽调整各元素位置
- ✅ **二维码生成**：自动生成邀请二维码
- ✅ **系统分享**：支持分享到微信、朋友圈等平台

### 1.2 模板类型
| 模板 | 背景色 | 风格 | 适用场景 |
|------|--------|------|----------|
| 经典典雅 | #FFF8E7 | 传统中式 | 正式场合 |
| 现代简约 | #F5F5F5 | 北欧风格 | 日常分享 |
| 极简黑白 | #1A1A1A | 黑白配色 | 文艺风格 |
| 喜庆红金 | #FF4444 | 红金配色 | 节日庆典 |

## 2. 海报生成验证

### 2.1 进入纪念册页面
1. 打开应用
2. 进入家族详情页
3. 点击"纪念册"入口
4. 或直接在底部导航找到纪念册入口

### 2.2 选择成员
1. 进入纪念册页面
2. 查看成员列表
3. 选择要包含的成员（建议 4-8 人）
4. 可使用"全选"快速选择

### 2.3 选择模板
1. 选择模板类型
2. 预览模板效果
3. 点击选择

### 2.4 自定义设置
1. 输入家族格言（可选）
2. 点击"下一步：自定义布局"

### 验证清单
- [ ] 成员选择正确显示
- [ ] 模板预览清晰
- [ ] 家族名称正确显示
- [ ] 格言正确显示

## 3. 拖拽布局调整验证

### 3.1 编辑模式
1. 点击"编辑布局"按钮
2. 进入编辑模式
3. 可看到所有可拖拽元素

### 3.2 拖拽操作

3. **删除元素**：长按元素，点击删除按钮

## 4. 二维码和分享验证

### 4.1 二维码功能
每个海报默认包含一个二维码，包含：
- 家族邀请链接
- 二维码标签"扫码加入家族"

### 4.2 分享功能
点击"分享海报"按钮后：

#### 分享选项
1. **保存到相册**：将海报保存为图片
2. **分享到微信**：直接分享到微信好友/朋友圈
3. **其他平台**：支持 QQ、微博等

#### 分享格式
- 格式：PNG/JPEG
- 分辨率：1080x1520（高分辨率）
- 文件名：`家族纪念册_YYYYMMDD.png`

### 4.3 模拟分享测试

#### 微信分享测试
1. 在设备上打开应用
2. 生成纪念册海报
3. 点击分享按钮
4. 选择"分享到微信"
5. 确认跳转微信成功
6. 检查海报显示正常

#### 朋友圈分享测试
1. 完成上述步骤
2. 选择"分享到朋友圈"
3. 确认跳转成功
4. 检查预览图显示

### 验证清单
- [ ] 二维码正确生成

- [ ] 分享按钮可点击

## 常见问题

### Q: 海报生成失败？
A:
1. 检查网络连接
2. 确认成员头像已加载
3. 重试生成
4. 查看错误日志

### Q: 二维码无法扫描？
A:
1. 确保二维码位置不超出边界
2. 检查二维码尺寸是否足够大
3. 确认分享的为原图而非压缩图

### Q: 拖拽时卡顿？
A:
1. 检查设备性能
2. 减少同时拖拽的元素数量
3. 退出编辑模式后重试

### Q: 分享失败？
A:
1. 检查微信是否安装
2. 确认应用有分享权限
3. 检查网络连接
4. 尝试保存到相册后手动分享

- **截图**：使用 expo-sharing 和 expo-file-system
- **分享**：调用系统原生分享
- **二维码**：预留接口，可接入 qrcode.react-native
- **拖拽**：使用 React Native PanResponder
