- **男性**: 淡蓝色 `#3B82F6`
- **女性**: 淡粉色 `#EC4899`
- **未知**: 灰色 `#9CA3AF`

### 连接线样式
- **颜色**: 古铜金 `#D4A574`
- **线宽**: 2px
- **形状**: 垂直→水平→垂直（平滑折线）
- **圆角**: 端点圆角处理

## 📱 运行测试

### 启动开发服务器
```bash
cd /workspace/FamilyTree
npm start
```

### 在设备上查看

#### iOS 模拟器
```bash
npm run ios
```

#### Android 模拟器
```bash
npm run android
```

#### Web 预览
```bash
npm run web
```

在浏览器中打开 `http://localhost:8081` 并导航到 `tree-demo` 标签页。

## 📝 手动测试执行步骤

2. 捏合靠近 → 观察缩小

## ⚠️ 注意事项

1. **性能**: 节点数量超过 50 个时建议添加虚拟化
2. **内存**: 频繁缩放可能导致内存占用增加
3. **深色模式**: 组件未针对深色模式优化
4. **可访问性**: 可添加 VoiceOver/TalkBack 支持
