# 合规验证指南：隐私政策与用户协议

## 概述

本指南提供家族树应用合规功能的完整验证方案，涵盖：
- 隐私政策页面（符合中国《个人信息保护法》）
- 用户协议页面
- 首次启动隐私弹窗
- 账号注销功能

### 3.4 验证步骤

#### 步骤 1：首次启动验证

```
1. 卸载应用（如已安装）
2. 重新安装应用
3. 启动应用
4. 观察是否弹出隐私政策弹窗
5. 验证结果：✅ 弹窗正确弹出
```

#### 步骤 2：勾选逻辑验证

```
1. 启动应用，等待弹窗出现
2. 不勾选复选框，点击"同意并继续"
3. 验证：按钮保持禁用状态
4. 勾选复选框
5. 验证：按钮变为可点击状态
6. 点击"同意并继续"
7. 验证：弹窗关闭，进入应用首页
```

#### 步骤 3：链接跳转验证

```
1. 启动应用，等待弹窗出现
2. 点击"隐私政策"链接
3. 验证：跳转到隐私政策页面
4. 返回弹窗
5. 点击"用户协议"链接
6. 验证：跳转到用户协议页面
```

#### 步骤 4：拒绝处理验证

```
1. 启动应用，等待弹窗出现
2. 点击"拒绝"按钮
3. 验证：弹窗关闭
4. 验证：应用显示受限状态或退出
```

## 5. 数据删除验证

### 5.1 Supabase 数据删除验证

#### 删除的数据表

| 表名 | 说明 | 删除条件 |
|------|------|---------|
| families | 家族信息 | created_by = userId |
| members | 成员信息 | created_by = userId |
| relationships | 关系记录 | created_by = userId |
| family_members | 家族成员关联 | user_id = userId |
| collaborations | 协作者记录 | user_id = userId |
| feedback_entries | 反馈记录 | user_id = userId |

#### 验证方法

```sql
-- 验证用户数据是否已删除
SELECT * FROM families WHERE created_by = 'user_id';
SELECT * FROM members WHERE created_by = 'user_id';
SELECT * FROM relationships WHERE created_by = 'user_id';
SELECT * FROM family_members WHERE user_id = 'user_id';
SELECT * FROM collaborations WHERE user_id = 'user_id';
SELECT * FROM feedback_entries WHERE user_id = 'user_id';

-- 预期结果：所有查询都返回空结果
```

### 5.2 账号删除验证

```sql
-- 验证用户账号是否已删除
SELECT * FROM auth.users WHERE id = 'user_id';

-- 预期结果：返回空结果
```

### 5.3 验证步骤

#### 步骤 1：准备测试数据

```
1. 使用测试账号登录
2. 创建至少1个家族
3. 添加至少3个成员
4. 建立至少1个关系
5. 添加至少1个协作者
```

#### 步骤 2：执行注销流程

```
1. 进入账号设置页面
2. 点击"注销账号"
3. 完成二次确认流程
4. 等待删除成功提示
```

#### 步骤 3：验证数据删除

```
1. 使用 Supabase Studio 登录
2. 查询相关数据表
3. 验证所有用户相关数据已删除
4. 验证用户账号已删除
```
