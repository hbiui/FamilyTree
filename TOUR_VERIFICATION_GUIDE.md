# 新手引导功能 - 验证清单

## 📋 功能概述

已实现的新手引导功能包括：

1. **三步新手引导流程**：
   - 欢迎页 - 介绍应用功能
   - 点击卡片查看详情
   - 长按节点添加子女/配偶
2. **自动触发**：首次进入家族树页面时自动显示
3. **持久化状态**：使用 localStorage 存储是否完成引导
4. **可重看**：完成引导后可点击按钮重新观看
5. **自定义蒙层**：半透明遮罩 + 动画气泡提示

## 🐛 已知问题与边界情况

### 边界情况一：存储禁用

```
情况：用户浏览器禁用 localStorage
处理：每次刷新都显示引导
验证：确保不报错，功能正常降级
```

### 边界情况二：快速切换页面

```
情况：在引导显示期间快速切换到其他页面
处理：确保状态正确清理，下次进入不显示（如果已完成）
```

### 边界情况三：引导进行时页面刷新

```
情况：在第二步时刷新页面
处理：应该判断 tourCompleted 状态，如果为 false 则重新显示第一步
```

```javascript
// localStorage 中的数据
{
  "state": {
    "tourCompleted": true,
    "currentStep": null,
    "isVisible": false
  },
  "version": 0
}
```

### 引导流程时间线

```
进入家族树页面
    ↓
检查 loading
    ↓ (loading 结束)
检查 tourCompleted
    ↓ (未完成)
检查是否有家族和成员
    ↓ (有数据)
300ms 延迟
    ↓
显示第一步引导
    ↓
用户交互 (点击下一步/跳过)
    ↓
显示第二步
    ↓
用户交互
    ↓
显示第三步
    ↓
用户点击"我知道了"
    ↓
标记 tourCompleted = true
    ↓
隐藏引导
```
